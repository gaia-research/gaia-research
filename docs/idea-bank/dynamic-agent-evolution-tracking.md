# Dynamic Agent Evolution Tracking

- **Rank:** 6
- **Viability:** Medium
- **Potential:** Very High

## What to research
- A living capability graph that updates when agents repeatedly demonstrate verified behaviors in real workloads.
- Thresholds for promotion, decay rules for stale capabilities, and anti-gaming protections.
- Signed push events from external systems so only approved capabilities become visible to the registry.

## Why it matters
- Turns the Skill Tree into a continuously refreshed map of real agent performance.
- Connects telemetry, benchmarking, and public proof into one long-term research program.
- Could become a defining differentiator if trust and governance are solved well.
